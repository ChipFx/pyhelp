View Modes
==========

SampleApp supports several view modes that control how your workspace is arranged.
Switch between them to match your workflow and screen size.

Available Modes
---------------

Standard
    The default layout. The canvas occupies the centre of the screen with
    tool panels docked on the left and right.

Fullscreen
    Hides all panels and toolbars.  Press :kbd:`F11` to toggle, or use
    **View → Fullscreen**.

Compact
    Combines the left and right panels into a single collapsible sidebar,
    giving the canvas more horizontal space.  Recommended on narrow screens.

Dual-Panel
    Splits the canvas area into two independent viewports.  Useful for
    comparing two areas of a large document side by side.

Switching Modes
---------------

You can switch modes from the **View** menu, or use these keyboard shortcuts:

* :kbd:`F11` — Toggle fullscreen
* :kbd:`Ctrl+Shift+C` — Compact mode
* :kbd:`Ctrl+Shift+D` — Dual-panel mode
* :kbd:`Ctrl+Shift+S` — Return to Standard mode

.. note::

   The current view mode is saved when you close the application and restored
   the next time you open it.  Your panel layout within each mode is also
   preserved separately.
